
#------------------------------------------------------------------------------

Clone Repo in /var/www

#------------------------------------------------------------------------------

.bashrc modifications (root):

alias bop='bash /var/www/maintenance/BOP-system-util.sh'

SOME COMMANDS ONLY WORK AS ROOT!

#------------------------------------------------------------------------------

/etc/ssh/sshd_config modifications:

ForceCommand /var/www/maintenance/BOP-system-util.sh

#------------------------------------------------------------------------------

Set up a static local IPv4

#------------------------------------------------------------------------------

Set up 2 Nginx virtual hosts in /etc/nginx/sites-available/default:

default
root /var/www/html/index.html

onion
root /var/www/html/onion-index.html

#------------------------------------------------------------------------------

Create a cronjob ($ crontab -e) and paste contents from /updates/cronjob.sh

#------------------------------------------------------------------------------

Install TOR:
https://support.torproject.org/apt/

#------------------------------------------------------------------------------

Configure TOR

nano /etc/tor/torrc

Uncomment these lines of config:

#HiddenserviceDir /var/lib/tor/hidden_service
#HiddenServicePort 80 127.0.0.1:80

#------------------------------------------------------------------------------

Paste public key, private key, hostname of BOP in /var/lib/tor/hidden_service/

Set up cloudflared
