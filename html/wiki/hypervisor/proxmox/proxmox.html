Proxmox

Proxmox is an open-source virtualization platform that supports both virtual machines (VMs) and containers. It features a web-based management interface, enabling users to create, manage, and monitor VMs and containers.

It's also possible to cluster multiple Proxmox servers, forming a Proxmox cluster. This clustering capability enables centralized management, high availability configurations, and efficient resource sharing across the connected nodes, enhancing the overall scalability and reliability of the virtualization environment.
