How to convert hexadecimal / decimal

Converting decimal to hexadecimal:

In the hexadecimal system, we have 16 numbers: Starting with the 0, 1, 2, 3, 4, 5, 6, 7, 9. When we come to 10, we start displaying the numbers as letters: A, B, C, D, E, F.

A=10, B=11, C=12, D=13, E=14, F=15.
