Simple Minecraft Server with Crafty

This method is tested on Debian 12.

We are going to set up a simple Minecraft server with a web interface for administration. After that we will make it accessable for other players.

We will have an unlimited amount of player slots and the ability to install plugins. The only restrictions would be the used hardware.

Installation

Step 1: Set up a debian server with a static local IPv4.

Step 2: Install docker and docker-compose. 


    # apt install docker-compose docker.io -y
