What is a Variable?

A variable is like a container that holds data, such as numbers, text, or other types of information. It has a name, a value, and a specific data type that defines the kind of data it can store. Variables are fundamental for storing and manipulating data in a program.

Integer Variables

Integers ( often referred to as Int ) hold whole numbers without any decimal points. For example, variables storing ages or counts would be of integer type. 


    int8: -128 to 127
    uint8: 0 to 255
    int16: -32,768 to 32,767
    uint16: 0 to 65,535
    int32: -2,147,483,648 to 2,147,483,647
    uint32: 0 to 4,294,967,295
    int64: -9,223,372,036,854,775,808 to 9,223,372,036,854,775,807
    uint64: 0 to 18,446,744,073,709,551,615

Floating-Point Variables

Floating-Point Variables These store numbers with decimal points, also known as floating-point numbers. Variables that need to hold values like measurements or percentages fall into this category.

Character Variables

Character Variables, often reffered to as "Char" hold single characters, like letters, numbers, or symbols. They are commonly used to store individual characters or small strings of text.

String Variables

These store sequences of characters, like words or sentences. They are used for storing longer text-based data.

Boolean Variables

Boolean Variables can have only two possible values: true or false / 1 or 0. They're used to represent binary states, such as on/off or yes/no conditions.

Array Variables

Array variables are used in programming to store multiple values of the same data type under a single name. They allow you to work with a collection of related data more efficiently. For example, if you have a list of numbers, you can store them in an array rather than using separate variables. Each value in an array has an index that represents its position. Arrays are helpful when dealing with groups of data that need to be processed together.

What is an If / else Statement?

In coding, "if/else" statements are used to make decisions based on certain conditions. They allow your program to perform different actions depending on whether a given condition is true or false. Here is a small example made in Python, because Python has easy syntax. In Python, the lines with "#" at the beginning are comments. Think of comments as a way to make notes and documentation in between your code. 


# Declaring an Integer Variable.
age = 15

# If / else Statement:
if age >= 18:
    print("You can vote!")
else:
    print("You are too young to vote"")
# IF the age is over 18, or 18, the script will print "You can vote!".
# If the age is under 18, it will print "You are too young to vote!".

What are Loops?

While Loops

A while loop is a programming construct that allows a certain block of code to be executed repeatedly as long as a specific condition is true. The loop keeps running until the condition becomes false. This type of loop is useful when you want to repeat an action multiple times based on a particular condition. Here is another simple example in Python: 


count = 1

while count <= 5:
    print(count)
    count += 1

In this example, the program uses a while loop to print "Hello!" five times. The loop starts with the variable "count" set to 1. It continues running as long as count is less than or equal to 5. Inside the loop, it prints "Hello!" and then increases count by 1. This process repeats until count reaches 6, when it is supposed to stop.

For Loops

A for loop helps you do something again and again. Imagine you want to count from 1 to 5 using a for loop. You would want to do it similar to this Python example: 


for number in range(1, 6):
    print(number)

In this example, the loop starts at 1 and goes up to 5. Each time, it prints the current number. For loops are handy when you want to repeat a task a specific number of times or go through a list of things. They save you time and make your code more organized.
