How to get started?

To get started with Coding / Programming, you don't need many things. A Computer, an Integrated Development Environment ( IDE ) or Text Editor and a compiler or interpreter ( depends on what Language you want to start with. For example: For the C Language you need a compiler, for Python you need an interpreter.

Nice to know: It is much better to learn the basics of a Language first, instead of instantly trying a big and complicated project. The results my not be as impressive, but with a complicated project as your first project, it is likely, that you start to doubt everything, and eventually give up.
