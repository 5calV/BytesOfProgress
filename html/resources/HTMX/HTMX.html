Use BytesOfProgress as your HTMX CDN

For many HTMX users it is common to load HTMX from a remote server or CDN. Now you can load your HTMX from BytesOfProgress. The advantage of this method is that the HTMX version is updated automatically, you don't even have to change the file name in the link as it will always stay the same.

BytesOfProgress always serves the up-to-date, unchanged version from GitHub.

All you have to do is load HTMX from this link: 


  https://bytesofprogress.io/resources/HTMX/htmx.min.js
