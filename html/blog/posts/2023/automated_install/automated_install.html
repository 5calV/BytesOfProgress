Automated Server Install

15th March 2023 / 12:15 PM

I wrote a small bash script which automatically installs Apache2 on a server and copies the HTML / CSS and Media files of this website to the Apache2 directory and then restarts Apache2. In case I have to move this blog to a different server I just have to execute this shell script and then attach the domain and SSL-Certificate to it.
