HomeLab Panel

06th December 2023 / 12:30 PM

For a few days I worked on a website which is a "homelab panel". It lets me access all of my selfhosted apps that are web-based or all my favorite sites quickly. Anyone can use this: I published it on GitHub!

It looks like this:

To use this for yourself, you can clone the repository and put the files on a webserver like Apache2 or Nginx. It works with 64x64 px logos. You can just edit the file's names, button texts and URLs in the HTML files to add your own logos / links. There is space for 12 logos / links in each category.
