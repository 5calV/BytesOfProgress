About this Website

This is a simple Website using HTML / CSS, which runs on an Apache2 / Debian server.

It was my intention to make the site that simple, because that way everybody is able to see all of the content, even with Javascript disabled in their browser.

Also, there is no need to make this site exceptionally fancy, it just has to contain the information I want to provide the visitors of this site.

BytesOfProgress will ALWAYS remain completely ad-free.
